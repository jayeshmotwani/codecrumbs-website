# Security

## Scope

This is a fully static site with no backend, no user data, no authentication, and no cookies. The attack surface is minimal by design.

## Reporting a vulnerability

If you spot something (e.g. a dependency with a known CVE, a malicious script somehow introduced), please report it privately rather than opening a public issue.

Email: **jayeshmotwani99@gmail.com**

Please include:
- What you found
- Steps to reproduce or a proof of concept
- Your assessment of severity

I'll respond within a few days. There's no bug bounty program, but I'll credit you in the fix commit if you'd like.
